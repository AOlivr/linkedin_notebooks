# linkedin_notebooks

Repositório com notebooks desenvolvidos como complemento aos conteúdos publicados no LinkedIn sobre séries temporais, sensoriamento remoto, ciência de dados e aprendizado de máquina.

A proposta é apresentar conceitos, métodos e aplicações práticas utilizando dados reais, desde a exploração inicial dos dados até a aplicação de técnicas estatísticas e de aprendizado de máquina.

Os exemplos abordam temas relacionados à análise de dados geoespaciais e sensoriamento remoto, buscando aproximar conceitos teóricos de aplicações práticas.
